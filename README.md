# NBA 知識挑戰

一個互動式的 NBA 知識問答遊戲，測試你對 NBA 球員、比賽規則和歷史紀錄的了解程度。

## 📖 專案介紹

這是一個為籃球社團活動設計的網頁遊戲，玩家需要回答 5 道隨機題目，測試自己的 NBA 知識。遊戲包含三種不同類型的題目，涵蓋球員識別、比賽規則和歷史紀錄等各個面向。

## ✨ 功能特點

- 🎮 **互動式問答遊戲**：選擇題形式的答題體驗
- 🎯 **三種題目類型**：猜球員題、規則題、歷史/紀錄題
- 🎲 **隨機出題**：每次遊戲都會從題庫中隨機選取 5 題，增加遊戲趣味性
- 📊 **即時反饋**：答題後立即顯示正確或錯誤，並顯示正確答案
- 🎁 **獎勵機制**：根據答對題數給予不同獎品
- 📱 **響應式設計**：適配不同螢幕尺寸，可在手機、平板和電腦上使用
- 🎨 **美觀介面**：採用漸層背景和現代化 UI 設計

## 🎯 題目類型

### 1. 猜球員題
- 根據球員照片和提示描述，選擇正確的球員
- 包含中英文對照的球員名稱
- 涵蓋現役和退役的知名 NBA 球員

### 2. 規則題
- 關於 NBA 比賽規則的選擇題
- 包含比賽時間、犯規規則、得分規則等
- 測試對籃球比賽規則的理解

### 3. 歷史/紀錄題
- 關於 NBA 歷史和紀錄的選擇題
- 包含球隊歷史、球員紀錄、總冠軍等
- 涵蓋從古至今的重要 NBA 里程碑

## 🎮 遊戲規則

1. 每位玩家一次挑戰 **5 題**
2. 題目會從題庫中**隨機選取**並打亂順序
3. 每題有 4 個選項，點選答案後會自動提交
4. 答對題目會立即顯示「正確！」提示
5. 答錯題目會顯示「錯誤！」並告知正確答案
6. 完成 5 題後會顯示最終結果和獲得的獎品

## 🎁 獎勵機制

根據答對題數給予不同獎品：

- **答對 5 題**：🥤 麥香紅茶 + 🍬 1 顆嗨啾
- **答對 4 題**：🥤 麥香紅茶
- **答對 1-3 題**：🍬 嗨啾（一題一顆，如答對 3 題則獲得 3 顆嗨啾）
- **答對 0 題**：很可惜，沒有獲得獎品

## 🚀 使用方式

### 方法一：直接開啟 HTML 檔案
1. 下載或複製整個專案資料夾
2. 確保所有檔案都在同一資料夾中
3. 雙擊 `index.html` 檔案，用瀏覽器開啟
4. 點擊「進入挑戰」按鈕開始遊戲

### 方法二：使用本地伺服器（推薦）
如果需要更好的體驗，建議使用本地伺服器：

```bash
# 使用 Python（Python 3）
python -m http.server 8000

# 或使用 Node.js（需要先安裝 http-server）
npx http-server -p 8000
```

然後在瀏覽器中開啟 `http://localhost:8000`

## 🔧 自訂題目

題目資料存放在 `topic.js` 檔案中的 `BasketballGame` 類別的 `initializeGame()` 方法中。

### 新增題目範例：

```javascript
{
    type: 'guess_player',        // 或 'rules' 或 'history_record'
    image: '球員照片.jpg',       // 僅猜球員題需要
    question: '這是題目內容',
    options: ['選項1', '選項2', '選項3', '選項4'],
    answer: 0,                   // 正確答案的索引（0-3）
    correct: '正確答案'          // 顯示的完整正確答案
}
```
